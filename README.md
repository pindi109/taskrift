# Taskrift

> Organize tasks. Own your day.

Taskrift is a focused, no-nonsense productivity tool built for individuals and small teams who want clarity without complexity. The brand feels sharp, calm, and empowering — like a quiet command centre for your work life. It targets ambitious professionals, indie hackers, and creators who value clean design and deep focus over bloated feature sets.

## Description
Build a to-do list app where users can create tasks, set due dates, mark them complete and organise into projects

## Tech Stack
| Layer | Technology |

## Getting Started

### Prerequisites
- Flutter SDK >= 3.0.0
- Firebase CLI
- A Firebase project (add `google-services.json` to `android/app/`)

### Installation
```bash
git clone https://github.com/pindi109/taskrift
cd taskrift
flutter pub get
flutter run
```

### Environment Setup
1. Create a Firebase project at [console.firebase.google.com](https://console.firebase.google.com)
2. Enable Authentication (Email/Password + Google)
3. Enable Firestore and Storage
4. Download `google-services.json` → place in `android/app/`
5. Download `GoogleService-Info.plist` → place in `ios/Runner/`

## Website
Static landing pages are in `/website/`. Deploy to Firebase Hosting, Netlify, or Vercel.
